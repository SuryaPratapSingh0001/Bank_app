# CLI Bank Management System

A simple Command Line Interface (CLI) based Bank Management System built using Core Java.  
This project demonstrates the use of Object-Oriented Programming (OOP) concepts in a real-world scenario.

## Features
- Opening Account
- Depositing
- Withdrawal
- Transfer
- Account Statements
- Listing Accounts
- Search Accounts by Customer Name
- Menu-driven CLI interface

## Concepts Used
- Core Java
- Object-Oriented Programming (Encapsulation, Inheritance, Polymorphism)
